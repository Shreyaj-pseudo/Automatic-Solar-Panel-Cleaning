Solar Panel Film Cleaner — System Architecture

Technical Reference · Subsystem Overview · Signal & Power Flow

SYS-01
⚡ Power Subsystem
🔌
AC-DC Supply
AC-DC 12V 5A
Primary wall power source
🔽
Buck Converter
20W Adjustable DC-DC
Steps 12V → 5V logic rail
🛡️
Dual Rails
12V motor / 5V logic
Isolated to prevent motor noise
SYS-02
📱 Input Subsystem
📱
Blynk IoT App
Blynk Platform (smartphone)
Virtual button → remote cycle start
📡
WiFi ESP8266
WiFi ESP8266 module
Receives Blynk signal over WiFi
Forwards command to Arduino
⏱️
60s Firmware Timer
millis() software timer
Auto-triggers every 60s
No physical input needed
SYS-03
🧠 Control Subsystem
💻
Microcontroller
Arduino Nano
Reads ESP8266, drives motor output
⚙️
Motor Driver
DRV8833 H-Bridge
Converts PWM → stepper current
🔵
Status LEDs
Red LED + Blue LED (as green)
🔴 Standby  ·  🔵 Running
⚡ ↓ 12V + 5V Power
📱 → WiFi Trigger Signal
🔀 ↓ PWM Drive Signal
SYS-04
⚙️ Drive Subsystem
🌀
Stepper Motor
Mini Stepper Nema 8
Precise low-speed film advance
⭕
Drive Pulley
4mm 15-Tooth Pinion
Motor-coupled belt engagement
🎚️
Step Rate Control
Firmware pulse frequency
Controls film advance speed
SYS-05
🎡 Mechanical Subsystem
📏
Timing Belt
20" / 100T 3/8" XL
Positive no-slip belt drive
⭕
Idler Roller
Adjustable tensioner
Belt tension & alignment control
🪛
Rigid Frame
Support structure
Prevents wrinkling & tracking error
SYS-06
📽️ Film Subsystem
🪟
Transparent Film Loop
High-transmittance plastic
Collects dust · light passes through
⬆️
Top Path (Active)
Film on panel surface
Sits on panel, collects dust
⬇️
Return Path (Underside)
Film under panel via slit
Gravity sheds dust at roller bend
◆ Operational Sequence Flow
📱
Cycle Triggered
Blynk app tap OR 60s firmware timer fires
→
🧠
MCU Activates
Arduino sends PWM pulses to DRV8833
→
🌀
Motor Turns
Nema 8 stepper spins pinion pulley · 🟢 LED on
→
📏
Belt Advances
XL timing belt pulls film across panel top
→
🔄
Film Returns
Dusty film loops under panel · gravity sheds dust
→
☀️
Light Restored
Clean film re-enters top · 🔴 LED returns to standby
Status Indicators:
Blue LED — Cycle actively running |
Red LED — Standby / awaiting Blynk or 60s timer
